# RADR — One-Page Concept Summary

**Recoilless Anti-Drone Rocket · 60 mm · Phase 0**  
**Version 1.7.0** · Notional — not procurement authority

---

## Problem

Squad and SOF face **Group 1–2 UAS** (FPV, quadcopters, loitering munitions, low-signature gliders) in the **150–1000 m** band. Machine guns lack reach; SAM is too heavy and scarce for every contact.

## Solution

**RADR** — a **lightweight, reusable**, shoulder-fired **recoilless** **mid-range drone destroyer**. One gunner, one reload: **≤ 9.0 kg** (launcher + round).

## How it works (7 steps)

1. Open breech · 2. Remove tube cap · 3. Load tube · 4. Close breech (locks)  
5. Hold **front trigger** → IR seeker + **lock tone** (retention stop releases)  
6. Pull **rear trigger** (front held) → fire  
7. Open breech → empty tube drops  

## Locked specs (summary)

| Item | Value |
|------|--------|
| Launcher | **40 in**, **≤ 5.5 kg** |
| Round | **60 mm × 18 in**, **≤ 3.5 kg** |
| Warhead | **300 × 7 mm** alloy cubes + **pyrotechnic dispersal** |
| Fuze | Radar/mm-wave **proximity** + **timed backup** @ ~20 ft |
| Motor | **2950–3050 N·s**, **~3.3 s** mildly progressive |
| Range goal | **1000 m** · **~330–350 m/s** at intercept (notional) |
| Backblast | **≤ 10 yd (30 ft)** rear |

## Safety

- **Rocket retention stop** — mechanical; releases only when breech closed + front held + lock tone  
- **Dual-trigger** — no fire without tone  
- **Gustav flip breech** — positive deadbolt lock
